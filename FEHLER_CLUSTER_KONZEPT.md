
# Fehler-Cluster und optimale Aufgabenpakete (Facetten-System)

## Prinzip: Gleiche Schwierigkeit, verschiedene Facetten

**Kernidee:** Statt verschiedene Schwierigkeitsgrade nacheinander, präsentieren wir **dieselbe kognitive Anforderung in verschiedenen Zahlen-Facetten**.

### Aufbau jeder A4-Seite:

```
FACETTE 1 (Zahlen-Variante A) → FACETTE 2 (Zahlen-Variante B) → FACETTE 3 (Zahlen-Variante C)
        ↓                              ↓                              ↓
   BASIS schaffen              ANWENDEN der Basis            VERKNÜPFEN mehrerer Konzepte
```

## Implementierungs-Prinzip

### Facetten-Matrix pro Fehlertyp:

| Schwierigkeitsstufe | Zehnerübergang | Stellenwert | Operation | Muster |
|---------------------|----------------|-------------|-----------|--------|
| **FACETTE 1: Basis** | Partnerzahlen | Zehner/Einer | Zeichen erkennen | Muster sehen |
| **FACETTE 2: Anwenden** | Zerlegung nutzen | Vergleichen | Beide Ops rechnen | Muster fortsetzen |
| **FACETTE 3: Verknüpfen** | Add+Sub verknüpfen | Reihenfolge wichtig | Umkehr-Kontrolle | Fehler finden |

### Code-Struktur:

```typescript
interface Facette {
  level: 'basis' | 'anwenden' | 'verknuepfen';
  numberVariants: number[]; // z.B. [7,3] oder [12,21]
  cognitiveLoad: 'same';    // Immer gleiche Schwierigkeit!
  paeckchenType: PaeckchenType;
}

interface ClusterWorksheet {
  title: string;
  facetten: [Facette, Facette, Facette]; // Genau 3 Facetten
  reflexion: string;
}

function generateFacettenWorksheet(errorCluster: ErrorCluster): ClusterWorksheet {
  return {
    title: errorCluster.name,
    facetten: [
      generateBasisFacette(errorCluster),      // Fundament legen
      generateAnwendenFacette(errorCluster),   // Transfer zeigen
      generateVerknuepfenFacette(errorCluster) // Zusammenhang stiften
    ],
    reflexion: errorCluster.reflexionsfrage
  };
}
```

## Didaktische Begründung

### Warum Facetten statt Schwierigkeitsstufen?

1. **Kognitive Konsistenz:** Das Kind muss nicht ständig "umschalten" zwischen leicht-mittel-schwer
2. **Tiefes Verständnis:** Dieselbe Denkoperation mit verschiedenen Zahlen → Abstraktion
3. **Entlastung:** "Ich kann das schon, nur mit anderen Zahlen" → Selbstvertrauen

### Warum Basis → Anwenden → Verknüpfen?

1. **FACETTE 1 (Basis):** Kernwissen explizit machen (Partnerzahlen, Stellenwert, Zeichen)
2. **FACETTE 2 (Anwenden):** Transfer in neue Situation (andere Zahlen, aber gleiche Logik)
3. **FACETTE 3 (Verknüpfen):** Mehrere Konzepte kombinieren (Addition+Subtraktion, Muster+Fehleranalyse)

### Lernpsychologische Vorteile:

- **Spiralprinzip:** Jede Facette greift die vorherige auf
- **Aktive Konstruktion:** Kind nutzt Basis-Wissen selbstständig
- **Metakognition:** "Wie habe ich das gemacht?" wird sichtbar

## Reflexions-Element auf jeder A4-Seite:

**Struktur:**
1. **Nach Facette 1:** "Was hast du entdeckt?"
2. **Nach Facette 2:** "Wie hast du das Wissen genutzt?"
3. **Nach Facette 3:** "Wie hängt alles zusammen?"

**Beispiel Cluster A (Zerlegung):**
- Facette 1 → "Welche Partnerzahlen kennst du?"
- Facette 2 → "Wie hilft dir 7+3=10 bei 7+5?"
- Facette 3 → "Warum kannst du mit derselben Zerlegung addieren UND subtrahieren?"
